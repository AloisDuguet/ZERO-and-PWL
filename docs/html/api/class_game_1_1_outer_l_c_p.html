Parameters

  • encoding: A vector of +1,-1 and 0 referring to which equations and variables are taking 0 value. +1 means equation set to zero, -1 variable, and zero none of the two

  • checkFeas: The component is added after ensuring feasibility, if this is true

  • custom: Should the components be pushed into a custom vector of polyhedra as opposed to OuterLCP::Ai and OuterLCP::bi

  • custAi: If custom polyhedra vector is used, pointer to the LHS matrix

  • custbi: If custom polyhedra vector is used, pointer to the RHS vector Given an encoding with +1, -1 and 0s optionally checks its feasibility, and adds the appropriate polyhedron for outer approximation to Game::OuterLCP::Ai and Game::OuterLCP::bi (or custAi and custbi).

    As a note to remember, 0 means, no branching is done on the said complementarity condition.
