class MathOpt::IP_Param : public MathOpt::MP_Param¶

This class handles parametrized integer programs.

Represents a Parameterized QP as

\[ \min_y c^Ty + (Cx)^T y \]
Subject to
\[\begin{split}\begin{eqnarray} Ay &\leq& b \\ y &\geq& 0 y_i &\in& &\mathbb{Z}&^{z_i} &\forall& i &\in& I \end{eqnarray}\end{split}\]

std::unique_ptr<GRBModel> solveFixed(arma::vec x, bool solve)¶

Given a value for the parameters \(x\) in the definition of IP_Param, solve the parameterized MIP program to optimality.

In terms of game theory, this can be viewed as the best response for a set of decisions by other players. solve decides whether the model has to be optimized or not
