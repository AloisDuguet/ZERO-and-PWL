Class Documentation¶

class Algorithms::Algorithm¶

The namespace Algorithms is responsible for the management of the algorithms that solve EPECs. Generally, the namespace is organized with multiple-level inheritances. The basic class is Algorithm, which implements some basic capabilities that all algorithms are sharing. Then, PolyBase managed the algorithm that either inner-approximate or full-enumerate the feasible region of each EPEC’s player (3rd level inheritors: e.g., Algorithm->PolyBase->FullEnumeration). The OuterApproximation class (2nd level inheritance) manages the outer approximation.

Subclassed by Algorithms::OuterApproximation, Algorithms::PolyBase
